# InVesalius 3: Marker Timestamp Patch

Цей репозиторій містить невелику модифікацію для відкритого медичного ПЗ **InVesalius 3**. Патч розширює базову модель 3D-маркерів, додаючи фіксацію точного часу їх створення (у форматі ISO 8601) для подальшої синхронізації просторових даних із зовнішніми медичними приладами.

## Структура
* `marker.py` — оновлений клас моделі з полем `timestamp`. Збережено повну зворотну сумісність із застарілими збереженнями `.inv3` (безпечне вилучення ключів).
* `test_marker.py` — автоматизовані юніт-тести для перевірки ініціалізації часу та безпечного парсингу legacy-файлів.

## Інтеграція та використання
1. Замініть оригінальний файл `invesalius/data/markers/marker.py` на модифіковану версію з цього репозиторію.
2. Жодних змін в інтерфейсі (GUI) робити не потрібно. Експорт маркерів у формат `.mkss` (CSV/TXT) автоматично підхопить нову колонку `Time` завдяки використанню динамічної рефлексії `dataclasses` в ядрі програми.

## Тестування

python tests/test_marker.py
