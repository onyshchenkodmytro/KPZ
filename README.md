# InVesalius 3: Marker Timestamp Patch

Цей репозиторій містить модифікацію для відкритого медичного ПЗ **InVesalius 3**. Патч розширює базову модель 3D-маркерів, додаючи фіксацію точного часу їх створення (у форматі ISO 8601) для подальшої синхронізації просторових даних із зовнішніми медичними приладами.

## Структура та зміни
* `invesalius/data/markers/marker.py` — оновлений клас моделі з полем `timestamp`. Збережено повну зворотну сумісність із застарілими збереженнями `.inv3` (безпечне вилучення ключів).
* `invesalius/gui/task_navigator.py` — модифіковано графічний інтерфейс користувача. До контролера таблиці маркерів (`marker_list_ctrl`) додано нову колонку "Time", яка безпечно парсить машинний формат ISO 8601 та виводить його у зручному для читання вигляді.
* `tests/test_marker.py` — автоматизовані юніт-тести для перевірки ініціалізації часу та безпечного читання legacy-файлів.

## Інтеграція та використання
1. Замініть оригінальні файли `marker.py` та `task_navigator.py` у вашій інсталяції InVesalius на файли з цього репозиторію.
2. Зовнішній експорт маркерів у формат `.mkss` (CSV/TXT) додаткових змін не потребує: нова колонка автоматично експортується завдяки використанню динамічної рефлексії (`dataclasses`) в ядрі програми.
